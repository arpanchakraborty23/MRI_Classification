# End TO End Mri Classification 


## How to run ?

### Create Conda Env
```bash
conda create -n env python==3.11 -y
```
```
conda activate env
```

### Install Dependencis
```bash
pip install -r requirements.txt
```

### Run Web app
```bash
python app.py
```

## Work Flow
1. Data Ingestion
2. Base Model
3. Model Train
4. Model Evaluation
5. Train Pipline
6. Prediction Pipline
